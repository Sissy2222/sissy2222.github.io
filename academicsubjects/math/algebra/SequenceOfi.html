Sequence Of i

What is the sequence of i?



dealing with imaginary numbers First of, i stands for 'imaginary number'...
a rather unfortunate name, yes

Let's write out our sequence
we know that i2 = -1, so let's built on that

i0 = 1
i1 = i
i2 = -1
i3 = (i1i)(2) = (i)(-1) = -i
i4 = (i2i)(2) = (-1)(-1) = 1
i5 = (i2i)(3) = (-1)(-i) = i
i6 = (i2i)(2)(2) = (-1)(-1)(-1) = -1
i7 = (i3i)(4) = (-i)(1) = -i
i8 = (i4i)(4) = (1)(1) = 1
i9 = (i4i)(5) = (1)(i) = i
i10 = (i5i)(5) = (i)(i) = i2 = -1
i11 = (i5i)(6) = (i)(-1) = -i

see the pattern...
